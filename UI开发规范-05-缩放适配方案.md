## 🎯 为什么需要缩放适配

### 问题场景

**设计稿分辨率：** 1920×1080（全高清）

**实际运行环境：**
- 开发电脑：1366×768（笔记本）
- 客户电脑：1280×1024（老式显示器）
- 工业触摸屏：1024×768
- 4K显示器：3840×2160

**不做缩放适配的后果：**

```
场景1：设计稿1920×1080，运行在1366×768
❌ 窗口太大，超出屏幕
❌ 右侧和底部被裁剪
❌ 用户看不到完整界面

场景2：设计稿1024×768，运行在1920×1080
❌ 窗口太小，只占屏幕一角
❌ 大量空白浪费
❌ 界面显得不专业
```

---

### 解决方案对比

#### 方案A：固定尺寸（不推荐）

```cpp
setFixedSize(1920, 1080);
```

**优点：**
- ✅ 简单，不需要额外代码

**缺点：**
- ❌ 无法适配不同分辨率
- ❌ 小屏幕显示不全
- ❌ 大屏幕浪费空间

---

#### 方案B：等比例缩放（推荐）⭐⭐⭐

```cpp
resize(1920, 1080);  // 初始大小
// 实现resizeEvent()和updateLayout()
// 所有元素等比例缩放
```

**优点：**
- ✅ 自动适配任意分辨率
- ✅ 保持界面比例不变形
- ✅ 用户可以调整窗口大小

**缺点：**
- ⚠️ 需要额外代码实现

---

## 📐 缩放适配原理

### 核心概念

#### 1. 基准分辨率

**定义：** 设计稿的画布尺寸，所有坐标都基于这个尺寸标注

**示例：**
```
基准分辨率：1024×768
按钮坐标：(100, 200, 120, 40)
```

---

#### 2. 实际分辨率

**定义：** 窗口当前的实际尺寸

**示例：**
```
用户调整窗口大小：800×600
或者：1280×960
```

---

#### 3. 缩放比例

**定义：** 实际分辨率与基准分辨率的比值

**计算公式：**
```cpp
scaleX = 实际宽度 / 基准宽度
scaleY = 实际高度 / 基准高度
scale = min(scaleX, scaleY)  // 取较小值，保持比例
```

**示例：**
```
基准分辨率：1024×768
实际分辨率：800×600

scaleX = 800 / 1024 = 0.78125
scaleY = 600 / 768 = 0.78125
scale = min(0.78125, 0.78125) = 0.78125
```

---

#### 4. 坐标转换

**公式：**
```
实际X = 基准X × scale
实际Y = 基准Y × scale
实际Width = 基准Width × scale
实际Height = 基准Height × scale
```

**示例：**
```
基准坐标：(100, 200, 120, 40)
缩放比例：0.78125

实际坐标：
X = 100 × 0.78125 = 78.125 ≈ 78
Y = 200 × 0.78125 = 156.25 ≈ 156
Width = 120 × 0.78125 = 93.75 ≈ 94
Height = 40 × 0.78125 = 31.25 ≈ 31
```

---

### 为什么取min(scaleX, scaleY)？

#### 情况1：scaleX = scaleY（等比例缩放）

```
基准：1024×768 (4:3)
实际：800×600 (4:3)

scaleX = 800/1024 = 0.78125
scaleY = 600/768 = 0.78125
scale = 0.78125

结果：完美适配，无黑边
```

---

#### 情况2：scaleX ≠ scaleY（宽高比不同）

```
基准：1024×768 (4:3)
实际：1280×720 (16:9)

scaleX = 1280/1024 = 1.25
scaleY = 720/768 = 0.9375
scale = min(1.25, 0.9375) = 0.9375

结果：
- 使用scaleY（较小值）
- 界面高度填满，宽度有黑边
- 保持4:3比例不变形
```

**图示：**
```
┌─────────────────────────┐
│         黑边            │
│  ┌─────────────────┐   │
│  │                 │   │
│  │   界面内容      │   │  ← 使用scale=0.9375
│  │   (4:3比例)     │   │
│  │                 │   │
│  └─────────────────┘   │
│         黑边            │
└─────────────────────────┘
      1280×720窗口
```

## 🛠️ 实现方案详解

### 完整代码实现

#### 头文件（mainwindow.h）

```cpp
#ifndef MAINWINDOW_H
#define MAINWINDOW_H

#include <QMainWindow>
#include <QLabel>

class MainWindow : public QMainWindow
{
    Q_OBJECT

public:
    MainWindow(QWidget *parent = nullptr);
    ~MainWindow();

protected:
    // 重写窗口大小改变事件
    void resizeEvent(QResizeEvent *event) override;

private:
    // 更新布局（缩放所有元素）
    void updateLayout();

    // 基准分辨率
    const int BASE_WIDTH = 1024;
    const int BASE_HEIGHT = 768;

    // 当前缩放比例
    double scale;

    // UI元素
    QLabel *lblBackground;
    QLabel *lblButton;
    // ... 其他元素
};

#endif // MAINWINDOW_H
```

---

#### 实现文件（mainwindow.cpp）

```cpp
#include "mainwindow.h"
#include <QResizeEvent>
#include <QtMath>  // qMin

MainWindow::MainWindow(QWidget *parent)
    : QMainWindow(parent)
    , scale(1.0)
{
    // 1. 设置窗口初始大小（可调整）
    resize(BASE_WIDTH, BASE_HEIGHT);
    setWindowTitle("缩放适配示例");

    // 2. 创建UI元素（使用基准坐标）
    lblBackground = new QLabel(this);
    lblBackground->setGeometry(0, 0, 1024, 768);
    lblBackground->setStyleSheet("background-color: #f0f0f0;");
    lblBackground->setScaledContents(true);

    lblButton = new QLabel(this);
    lblButton->setGeometry(100, 200, 120, 40);
    lblButton->setStyleSheet("background-color: #0078d7; color: white;");
    lblButton->setAlignment(Qt::AlignCenter);
    lblButton->setText("按钮");
    lblButton->setScaledContents(true);

    // 3. 初始化布局
    updateLayout();
}

MainWindow::~MainWindow()
{
}

// 窗口大小改变事件
void MainWindow::resizeEvent(QResizeEvent *event)
{
    QMainWindow::resizeEvent(event);
    updateLayout();  // 更新布局
}

// 更新布局
void MainWindow::updateLayout()
{
    // 1. 计算缩放比例
    double scaleX = (double)width() / BASE_WIDTH;
    double scaleY = (double)height() / BASE_HEIGHT;
    scale = qMin(scaleX, scaleY);  // 取较小值

    // 2. 更新所有元素的位置和大小
    // 公式：实际坐标 = 基准坐标 × scale
    
    lblBackground->setGeometry(
        0 * scale,           // X
        0 * scale,           // Y
        1024 * scale,        // Width
        768 * scale          // Height
    );

    lblButton->setGeometry(
        100 * scale,         // X
        200 * scale,         // Y
        120 * scale,         // Width
        40 * scale           // Height
    );

    // ... 其他元素同理
}
```

---

### 代码详解

#### 1. 初始化

```cpp
MainWindow::MainWindow(QWidget *parent)
    : QMainWindow(parent)
    , scale(1.0)  // 初始缩放比例为1.0
{
    resize(BASE_WIDTH, BASE_HEIGHT);  // 设置初始大小
    // 注意：使用resize()而不是setFixedSize()
    // resize()允许用户调整窗口大小
}
```

---

#### 2. 创建元素

```cpp
// 使用基准坐标创建元素
lblButton = new QLabel(this);
lblButton->setGeometry(100, 200, 120, 40);  // 基准坐标
```

**重要：** 创建时使用基准坐标，不要乘以scale！

---

#### 3. resizeEvent()

```cpp
void MainWindow::resizeEvent(QResizeEvent *event)
{
    QMainWindow::resizeEvent(event);  // 调用父类方法
    updateLayout();  // 更新布局
}
```

**触发时机：**
- 窗口创建时
- 用户拖动窗口边框调整大小时
- 最大化/还原窗口时

---

#### 4. updateLayout()

```cpp
void MainWindow::updateLayout()
{
    // 步骤1：计算缩放比例
    double scaleX = (double)width() / BASE_WIDTH;
    double scaleY = (double)height() / BASE_HEIGHT;
    scale = qMin(scaleX, scaleY);

    // 步骤2：更新所有元素
    lblButton->setGeometry(
        100 * scale,   // 基准X × scale
        200 * scale,   // 基准Y × scale
        120 * scale,   // 基准Width × scale
        40 * scale     // 基准Height × scale
    );
}
```

**注意：** 每个元素都需要更新！

---

### 批量更新优化

#### 方法1：使用结构体数组

```cpp
// 头文件
struct ElementInfo {
    QLabel *widget;
    int baseX, baseY, baseWidth, baseHeight;
};
QVector<ElementInfo> elements;

// 构造函数
ElementInfo info;
info.widget = lblButton;
info.baseX = 100;
info.baseY = 200;
info.baseWidth = 120;
info.baseHeight = 40;
elements.append(info);

// updateLayout()
void MainWindow::updateLayout()
{
    double scaleX = (double)width() / BASE_WIDTH;
    double scaleY = (double)height() / BASE_HEIGHT;
    scale = qMin(scaleX, scaleY);

    for (const ElementInfo &info : elements) {
        info.widget->setGeometry(
            info.baseX * scale,
            info.baseY * scale,
            info.baseWidth * scale,
            info.baseHeight * scale
        );
    }
}
```

**优点：**
- ✅ 代码简洁
- ✅ 易于维护
- ✅ 添加新元素只需append

---

#### 方法2：使用宏定义

```cpp
// 定义宏
#define SCALE_GEOMETRY(widget, x, y, w, h) \
    widget->setGeometry(x * scale, y * scale, w * scale, h * scale)

// updateLayout()
void MainWindow::updateLayout()
{
    double scaleX = (double)width() / BASE_WIDTH;
    double scaleY = (double)height() / BASE_HEIGHT;
    scale = qMin(scaleX, scaleY);

    SCALE_GEOMETRY(lblBackground, 0, 0, 1024, 768);
    SCALE_GEOMETRY(lblButton, 100, 200, 120, 40);
    SCALE_GEOMETRY(lblSidebar, 0, 80, 200, 688);
    // ... 其他元素
}
```

**优点：**
- ✅ 代码更简洁
- ✅ 减少重复代码

## ⚠️ 常见问题与解决方案

### 问题1：窗口变形

**现象：**
```
基准：1024×768 (4:3)
实际：1280×720 (16:9)
界面被拉伸变形
```

**原因：** 使用了scaleX和scaleY分别缩放

**错误代码：**
```cpp
lblButton->setGeometry(
    100 * scaleX,   // ❌ 使用scaleX
    200 * scaleY,   // ❌ 使用scaleY
    120 * scaleX,
    40 * scaleY
);
```

**正确代码：**
```cpp
scale = qMin(scaleX, scaleY);  // ✅ 统一使用scale
lblButton->setGeometry(
    100 * scale,
    200 * scale,
    120 * scale,
    40 * scale
);
```

---

### 问题2：图片模糊

**现象：** 窗口缩放后，图片变模糊

**原因：** 图片被拉伸或缩小，产生失真

**解决方案A：使用高分辨率图片**
```
设计稿：1024×768
图片尺寸：2048×1536 (2倍)

缩小时：清晰
放大时：相对清晰
```

**解决方案B：使用SVG矢量图**
```cpp
// 使用SVG图标
QIcon icon("icon.svg");
button->setIcon(icon);
```

**优点：**
- ✅ 任意缩放不失真
- ✅ 文件小

**缺点：**
- ⚠️ 不适合复杂图片（如照片）

---

### 问题3：字体太小

**现象：** 窗口缩小后，文字看不清

**解决方案：动态调整字体大小**

```cpp
void MainWindow::updateLayout()
{
    double scaleX = (double)width() / BASE_WIDTH;
    double scaleY = (double)height() / BASE_HEIGHT;
    scale = qMin(scaleX, scaleY);

    // 更新元素位置
    lblButton->setGeometry(100 * scale, 200 * scale, 120 * scale, 40 * scale);

    // 动态调整字体大小
    QFont font = lblButton->font();
    font.setPixelSize(14 * scale);  // 基准字体14px
    lblButton->setFont(font);
}
```

---

### 问题4：性能问题

**现象：** 拖动窗口边框时卡顿

**原因：** updateLayout()频繁调用，计算量大

**解决方案A：减少重绘次数**

```cpp
void MainWindow::resizeEvent(QResizeEvent *event)
{
    QMainWindow::resizeEvent(event);
    
    // 使用定时器延迟更新
    static QTimer *timer = new QTimer(this);
    timer->setSingleShot(true);
    timer->setInterval(50);  // 50ms延迟
    
    disconnect(timer, &QTimer::timeout, this, &MainWindow::updateLayout);
    connect(timer, &QTimer::timeout, this, &MainWindow::updateLayout);
    
    timer->start();
}
```

**解决方案B：优化计算**

```cpp
// 缓存计算结果
void MainWindow::updateLayout()
{
    double scaleX = (double)width() / BASE_WIDTH;
    double scaleY = (double)height() / BASE_HEIGHT;
    double newScale = qMin(scaleX, scaleY);
    
    // 只有缩放比例变化时才更新
    if (qAbs(newScale - scale) < 0.001) {
        return;
    }
    
    scale = newScale;
    
    // 更新元素...
}
```

---

### 问题5：黑边问题

**现象：** 窗口宽高比与设计稿不同时，出现黑边

**原因：** 使用qMin(scaleX, scaleY)保持比例

**解决方案A：接受黑边（推荐）**
```
优点：界面不变形
缺点：有黑边
适用：大多数情况
```

**解决方案B：填充黑边**
```cpp
// 设置窗口背景色
setStyleSheet("background-color: #000000;");

// 或者使用背景图片
QPalette palette;
palette.setBrush(QPalette::Window, QBrush(QPixmap("background.png")));
setPalette(palette);
```

**解决方案C：强制宽高比（不推荐）**
```cpp
void MainWindow::resizeEvent(QResizeEvent *event)
{
    // 强制保持4:3比例
    int newWidth = event->size().width();
    int newHeight = newWidth * 3 / 4;
    
    if (newHeight != event->size().height()) {
        resize(newWidth, newHeight);
        return;
    }
    
    QMainWindow::resizeEvent(event);
    updateLayout();
}
```

**缺点：**
- ❌ 用户无法自由调整窗口
- ❌ 体验不好

## 🚀 进阶技巧

### 技巧1：最小/最大窗口尺寸

```cpp
MainWindow::MainWindow(QWidget *parent)
    : QMainWindow(parent)
{
    resize(1024, 768);
    
    // 设置最小尺寸（防止窗口太小）
    setMinimumSize(640, 480);
    
    // 设置最大尺寸（可选）
    setMaximumSize(1920, 1080);
}
```

---

### 技巧2：保持宽高比

```cpp
void MainWindow::resizeEvent(QResizeEvent *event)
{
    // 计算目标宽高比
    const double targetRatio = (double)BASE_WIDTH / BASE_HEIGHT;
    
    int newWidth = event->size().width();
    int newHeight = event->size().height();
    double currentRatio = (double)newWidth / newHeight;
    
    // 如果宽高比偏差超过5%，调整窗口大小
    if (qAbs(currentRatio - targetRatio) > 0.05) {
        if (currentRatio > targetRatio) {
            // 窗口太宽，调整宽度
            newWidth = newHeight * targetRatio;
        } else {
            // 窗口太高，调整高度
            newHeight = newWidth / targetRatio;
        }
        resize(newWidth, newHeight);
        return;
    }
    
    QMainWindow::resizeEvent(event);
    updateLayout();
}
```

---

### 技巧3：平滑缩放动画

```cpp
#include <QPropertyAnimation>

void MainWindow::smoothResize(int targetWidth, int targetHeight)
{
    QPropertyAnimation *animation = new QPropertyAnimation(this, "size");
    animation->setDuration(300);  // 300ms动画
    animation->setStartValue(size());
    animation->setEndValue(QSize(targetWidth, targetHeight));
    animation->setEasingCurve(QEasingCurve::InOutQuad);
    animation->start(QAbstractAnimation::DeleteWhenStopped);
}

// 使用
void MainWindow::onFullscreenClicked()
{
    smoothResize(1920, 1080);
}
```

---

### 技巧4：记住窗口大小

```cpp
#include <QSettings>

// 保存窗口大小
void MainWindow::closeEvent(QCloseEvent *event)
{
    QSettings settings("MyCompany", "MyApp");
    settings.setValue("windowSize", size());
    QMainWindow::closeEvent(event);
}

// 恢复窗口大小
MainWindow::MainWindow(QWidget *parent)
    : QMainWindow(parent)
{
    QSettings settings("MyCompany", "MyApp");
    QSize savedSize = settings.value("windowSize", QSize(1024, 768)).toSize();
    resize(savedSize);
}
```

---

### 技巧5：多显示器支持

```cpp
#include <QScreen>
#include <QApplication>

MainWindow::MainWindow(QWidget *parent)
    : QMainWindow(parent)
{
    // 获取主屏幕分辨率
    QScreen *screen = QApplication::primaryScreen();
    QRect screenGeometry = screen->geometry();
    
    int screenWidth = screenGeometry.width();
    int screenHeight = screenGeometry.height();
    
    // 窗口大小为屏幕的80%
    int windowWidth = screenWidth * 0.8;
    int windowHeight = screenHeight * 0.8;
    
    resize(windowWidth, windowHeight);
    
    // 居中显示
    move((screenWidth - windowWidth) / 2, 
         (screenHeight - windowHeight) / 2);
}
```

# 国能机器人巡检系统 - 编译说明

## 项目文件
- mainwindow.h - 主窗口头文件
- mainwindow.cpp - 主窗口实现文件
- main.cpp - 程序入口
- untitled.pro - Qt 项目配置文件

## 编译方法

### 方法1：使用 Qt Creator（推荐）
1. 打开 Qt Creator
2. 点击 "文件" -> "打开文件或项目"
3. 选择 `untitled.pro` 文件
4. 点击左下角的绿色"运行"按钮（或按 Ctrl+R）

### 方法2：使用命令行
确保 Qt 的 bin 目录在系统 PATH 中，然后执行：
```bash
cd d:\qttest\untitled
qmake untitled.pro
nmake          # Windows MSVC
# 或
mingw32-make   # Windows MinGW
# 或
make           # Linux/Mac
```

## 界面说明

### 整体布局
- **窗口大小**：1200x800（可缩放）
- **配色方案**：深蓝色科技工业风（#1A365D）

### 顶部标题栏（60px高）
- 左右横线 + 中间标题："国能xxx机器人巡检系统"
- 右侧三个按钮：参数设置、系统配置、退出系统

### 左侧导航栏（220px宽）
- 实时监控
- 历史查询
- 故障诊断
- 数据统计
- 地图管理
- 巡检策略

### 右侧内容区
- 顶部显示当前位置
- 白色背景内容展示区

### 底部状态栏（30px高）
- 显示系统运行状态

## 功能特性
1. 点击左侧导航按钮，右侧会显示对应的"当前位置"
2. 点击"退出系统"按钮会关闭程序
3. 所有样式使用 QSS 实现，易于修改
4. 使用 Qt 标准控件，无第三方依赖

## 技术要点
- 使用 QFrame::HLine 实现标题栏的左右横线
- 使用 QSS 样式表统一管理界面风格
- 信号槽机制实现按钮交互
- 布局管理器实现响应式界面
